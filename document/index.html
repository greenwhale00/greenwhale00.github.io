    5. display

    1. display none ; 없앰(가장중요)

      display flex ; 박스 가로배열

      display block ; 나타냄, 표현됨, box 속성 줌

      display inline-block ; box의 속성을 가지고 inline의 속성도 가진다(가로배열, 띄어쓰기, vertical-align)

      display inline; box 속성 없앰

    position

      position : 겹쳐놓기(좌표를 가진다)

      position:satic; 포지션이 아니게 한다

      position:relative; 이어받는 기준

      position:absolute; 겹쳐놓는다(좌표를 가지고 떠오른다/쪼그러든다-기준이 X, body가 기준)

      absolute top,left,botton,right

      position:fixed; 기준을 잡아줄 수 X(기준자체가 X)-화면을 기준으로 한다. 화면 바깥으로 나가도 스크롤바가 생기지 않는다.

      z-index; 겹침의 순서를 바꾼다
