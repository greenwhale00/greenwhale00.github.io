display & position

  • display 01

    inline / block / inline-block

    inline 요소는 width height, 상하margin, padding 값을 줄 수가 없다
    inline-block 요소의 부모에 text-align:center;로 박스를 가로로 배열할 수 있다.
    inline-block 요소 사이의 간격을 없애려면 (박스를 이어붙이려면) font-size:0;을 해 주어야 한다.
    inline요소를 block요소로 바꿈
    박스 사이의 간격이 4px정도 생긴다 폰트를 살리려면 inline-block요소에 다시 font-size를 넣어줘야한다.

  • position 01

    position: relative, absolute, fixed;

    position:relative; 좌표를 가지는 position상태가 된다.
    일반적인 positon의 floating이나 width의 변화가 없다.
    position:absolute; 뜬다/쪼그라든다/기준점이 없어져서 body를 기준으로 포개진다.
    block의 속성을 가지게 된다.
    position: absolute; 의 기준을 부모요소의 position: relative, absolute, fixed로 기준이 된다.
    position:absolute; 뜬다/쪼그라든다/기준점이 없어져서 body를 기준으로 포개진다.
    block의 속성을 가지게 된다.
    position-fixed는 기준이 없다 (viewport를 기준으로 한다) 스크롤바가 생기지 않는다
    z-index로 흐름의 순서를 바꿔줄 수 있다. (설정값은 정수를 쓴다)

Hover를 이용한 Position 이해

  • Hover 01

    Hover 기능을 이용해 다양한 꾸밈기능을 넣어보았다

    nav menu :

    line-height는 50px로 띄우고 border-top/bottom으로 전체 박스에 윤곽선 넣기
    nav>ul 잡아서 width는 1200px, margin: 0 auto;로 가운데 정렬, display:flex;로 5개 가로배치
    nav>ul>li 잡아서 border-left로 메뉴박스 사이 윤곽선을 넣는다
    menu05번 박스 왼쪽에서 선을 그어줘야 하기 때문에 nav>ul>li:last-child를 잡아서 border-left선을 한번 더 긋는다.
    menu0$에 hover하면 박스 밑에 선이 그이도록 nav>ul>li::after를 잡아준다
    nav>ul>li::after에는 무조건 content:"";을 넣어주고 바닥쪽에 선이 생기게 bottom:0;잡고 height로 굵기 조절. 중간에서 선이 생기게 left: 50%; transform: translateX(-50%); 넣어준다.
    떠올라야 하므로 position: absolute;/nav>ul>li가 기준이 되게 position: relative;
    hover 되기 전 after에 width는 0, transisition은 적절히 0.5s
    nav>ul>li:hover::after에 width를 100%줘서 보이게 한다!!
    메뉴 박스 li를 display:block;시켜 박스 전체에 커서가 올라갈때 손가락모양이 되도록 함

    section class = hover01

    section 밑 container에
    display: flex; gap: 30px; width: 1200px; margin: 0 auto; 넣어서 container아래 피규어에 들어갈 사진 가로 배치
    figure밑에 이미지 넣고 각 이미지 밑에 box div넣어서 이미지를 덮는다.
    box는 fighre 이미지를 덮을수 있게 top: 0;, left: 0;, width: 0;, height: 100%; 값으로 좌표 주고 position: absolute;으로 떠오르게 함.
    (반드시 box의 부모요소인 figure에 position: relative;줘야함.)
    box에 쓰인 hover text는 display: flex;, justify-content: center;, align-items: center;로 박스 중앙에 오게 함
    hover 되면 box의 이미지는 옆으로 움직이여야 하므로 .hover01 .container figure:hover img 잡아서 transform: translateX(50px); 시킨다.
    (이러면 이미지가 figure 바깥으로 빠져 나가기 때문에 figure에서 overflow: hidden;을 넣어줘야함)
    자연스럽게 움직여야 하므로 hover 전 .hover01 .container figure img를 잡아서 transition: transform 0.5s; 시킨다
    이미지를 덮어쓴 박스는 hover되었을때 왼쪽에서 나타나게끔 .hover01 .container figure:hover .box를 잡고 width: 150px;, opacity: 1; 값을 줘서 보이게 한다.

    section class = hover02

    linear-gradient를 사용해 hover02 section 전체에 백그라운드를 넣었다.

    hover01과 박스, 이미지 배열 상동

    hover되었을때 figure박스 1.좌+상/2.하+우에 선이 그이도록 한다.

    1.좌+상 = .hover02 .container figure::before
    border-top/left로 선 긋기.
    top: 0;, left: 0; 좌표 주고 position: absolute; (.hover02 .container figure에 기준 position잡는다)
    width: 0;, height: 0;,box-sizing: border-box;, opacity: 0;으로 hover전 안보이게 함
    2. 하+우 = .hover02 .container figure::after
    1번과 동일. BUT!! 좌+상 선이 다 그여진 다음에 그여져야 하므로 transition: 0.5s 0.3s(여유시간 더 주기);

    1.좌+상/2. 하+우 모두 hover 주기
    width: 100%;, height: 100%;, opacity: 1;

    section class = hover03

    hover02와 유사하지만 사각형의 사변의 선이 순차적으로 그려진다

    1. left to right - .hover03 figure::before
    width: 0;, height: 10px; -> (hover) width: 100%

    2. right to bottom - .hover03 figure::after
    width: 10px;, height: 0; -> (hover) height: 100%

    3. bottom to left - .hover03 figure .box::before
    width: 0;, height: 10px; -> (hover) max-width: 100%

    4. left to top - .hover03 figure .box::after
    width: 10px;, height: 0; -> (hover) height: 100%
